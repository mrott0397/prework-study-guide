HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML is used to create the structure of a webpage. It is made up pf elemts which work together to provide content
  • HTML elements are usually composed of an opening and closing tag, represented with angle brackets
  • Elements included within the head are not visible when a user looks at a webpage
  • All the elements that we want the user to see and interact with must be placed inside the body element
  • DOCTYPE html is the Decloration at the top. This decloration tells the browser what document type it should expect when opened
  • html elements are case sensitive
  • "html lang="en" is the start of the document and telling the broswer the language is English (en) This is an attribute
  • An HTML attribute defines additional information about an element. HTML attributes are always defined in the beginning element, not the closing element
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata such as character set, description, keywords, author and viewport
  • "meta charset= "UTF-8" The meta elemt defines the charset, or character set, as UTF-8, which allows us to use most characters and symbols
  • The title element sets the title of the webpage
  • An HTML develepor is not only responsible for providing the structure for thr webpage, but also making sure that it provides a positive user experience
  • Elements that are nested inside other elements are called CHILD ELEMENTS. If there are other elements nested inside of a body element, those three nested elements are known as child elements
  • The element that holds all the children elements, is the PARENT ELEMENT
  • In Semantic HTML Elements, the name of the element describes the content and the roll it plays in a webpage
  • The p element represents a paragraph or block of text
  • The ul element represents Unordered Lists
  • The ol element represents Ordered Lists
  • The li element represents List Items
  • The img element can contain various attributes; the src attribute is the most important because it defines the location of the image
  • The br element creates a line of empty space, or line break
  • The a element (which stabds for anchor) creates links to the same webpage or others. The href attribute points to the url for the link
  • The alt attribute contains a text string that describes the apperance and functionality of the image. The alt text is important for screen readers and other assistive technologies. Alt text can also improve search engine rankings
  • The footer element appears at the bottom of the page and usually contains the author, copywrite, contacts, site map, and navigation
  • Section elements are semantic elements that define a section in a document that contains thematic content
  • An important takeaway is that we can use semantic HTML elements to indicate the roll of their content, as well as to give structure to the content

CSS

  • To add style to your webpage, you use Cascading Style Sheets (CSS) which is a programming language used by developers to define how the webpage content is presented to users.
  • CSS defines the colors, aesthetics, visual layout, and orginization
  • There are three ways to style a webpage using CSS: Inline CSS, Internal CSS Style Sheet, External CSS Style Sheet.
  • When a developer wants to style an element directly in the HTML file, they use Inline CSS.
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • Inline styles are useful for quick and permanent changes but are less flexible than external and internal style sheets, because each inline style must be edited individually whenever you make a design change.
  • An ecample of inline css is using the style attribute to change the color of a single h1 heading: h1 style= "color:red;"
  • It is possible to embed an entire style sheet directly into the HTML file using a style element in the head element. This internal style sheet contains CSS rules for the webpage in the head section of the HTML file. The rules only apply to that page, and you can configure CSS classes and ID's.
  • A single change to the CSS rule with apply to all the tagged page elements.
  • The most common way to style a webpage is by usiong an external CSS style sheet.
  • External CSS style sheet is preffered in most cases because it allows developers to keep all their CSS rules in a seperate file, which makes design changes easier.
  • When using an external style sheet, we link the file using a link element placed in the HTML's head.
  • Seperation of concerns is a widely used term in web development and refers to seperating code into different filed based on use. This is based on the idea that each section of code should have its own responsibility.
  • A selector defines the element or attributes to which the rules, or declarations will apply.
  • Declarations contain two important components: 1. The CSS property we want to apply 2. Value of the property. For example: The declaration color:blue; includes the CSS property COLOR and the value for that property, BLUE.
  • The margin property acts much like a margin in a word proccesing document. A margin property indicates how much space we want around the outside of an element.
  • The margin is not visible on the page; it merely adds a buffer
  • When we set the margin property to 0, we are indicating that we want NO extra space.
  • The padding property adds a space around the content inside an element. When you increase padding, you make the entire element bigger.
  • *: To apply rule to all visible elements.
  • Section elements are semantic elements that define a section in a document that contains thematic content
  • Do Not Repeat Yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained.
  • text-align: One way to center a title or heading.
  • display:block; : the display property is assigned with the block value, which assigns each iage to start a new line.
  • fontsize
  • Add CSS file to head section with meta.
  • rel: Specifies the relationship between the current document and the linked document source.
  • href: Specifies the location (URL) of the external resource. href is using a relative path, this means the location is derived fromm the current location.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector. Then the class attributes can be assigned to the element in the HTML, enabling us to freely choose which elements will be assigned the CSS rule.
  • Class selector: .card
  • .card: The peroid preceding card designates this as a class selector. Any HTML element assigned to the class, will gaine the property assignments from the CSS.
  • The box-shadow CSS property adds shadow effects around an element frame. You can set multiple effects seperated by commas.
  • A box shadow is described by X and Y offsets relative to the element, blur, and spread radius, and color.
  • Assigning colors in CSS can be done in many ways, including, by semantic refrences such as gray or blue. We can also refer to a color by its hexadecimal code, or hex #

JavaScript

  • A Variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script
  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites.
  • Using JavaScript, you can power your websites with effects, interactivity, data communication, and more.
  • Browser Application Programming INterfaces (API's) built into web browsers, providing functionality such as dynamically creating HTML and setting CSS styles; Collecting and manipulating a video strea from a users webcvam, or generating 3D graphics and audio samples.
  • Third-party API's that allow developers to incorperate functionality in sites fromo other content providers such as Twitter and Facebook.
  • In JavaScript, to declare a variable, we use the var keyword. After the var keyword, we give the variable name. This name must be unique because it is how we will refrence the information stored in the variable in our code.
  • We use an assignment operator = to assign the specific data that we want stored in our variable. Example: var shape = "circle"
  • All strings are surrounded by qoutation marks.
  • Each line of JavaScript ends with a semicolon
  • console.log: We can use the console.log method to output a message to the web console by adding an argument
  • console.log(topics); : In this case, console.log will output the value of topics to the web console because we've added it as an argument inside the parentheses
  • The word console refers to a test environment thats used to check out a code. .log is a method, which means a set of instructions that can be executed by a computer.
  • In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom.
  • An if statement is a way of introducing decision-making into our code. When using the if statement, code will only execute if the statement meets conditions, or is true.
  • When the computer encounters an if statement in a JavaScript file, it will check to see if the condition that we provide is truthy. If it is, then it will do whatever action we ask it to do in between the curly brackets. If the condition isn't true or has a value that isn't truthy, then the computer will ignore whatever is in between the curly brackets and move on.
  • In JavaScript, we can have more than one condition by using else if.
  • The computer will continue through all of the conditions that we list, evaluating whether or not a condition is truthy. Once it finds a condition that is truthy, it will run the code in the curly brackets associated with that condition, and then break out of the if statement
  • === : the strict equality operator checks to see if two values are equal, and returns a Boolean result true if they are equal and false if they are not
  • An Array is a single variable that is used to hold a group of data.
  • Arrays are typically used to hold data that is related in some way.
  • The following variable declaration demonstrates how we would represent our group of shapes in an array named shapes: var shapes = ["triangle", "pentagon", "square", "circle"];
  • Arrays are not limited to contain only a single type of data. A single array can store different types of data, such as strings, numbers, and so on.
  • Following example has info about a student named Lu; name age, and whether he is enrolled: var studentInfo = ["Lu", 54, true]; Lu is a string, needs quotation marks, 54 is a number, true i a Boolean value
  • To work with individual data items inside an array, we need one more piece of information: the location of the data item in the array. This location is identified with a unique number called an index.
  • In JavaScript and other programming languages, arrays are zero-indexed and sequential. This means that the first item in any array has an index of 0, not 1.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over. Developers must use another statement, such as the conditional if statement
  • By using the for loop, we can tell the computer to go through each of the shapes (items) in our array and log them to the console until there are none left.
  • In order to use the for loop, we need the shapes(item) index and the arrays length. The array length is important because the for loop needs to know how many times the code will run until there are no items left in the array.
  • We can find the length of the array by using a built-in property of an array, named length.
  • A for loop contains the following thre important statements: 1.The first statement determines the starting point for our loop. Since we want to start at the beginning of the array, we declare a variable x and give it a value of 0 (because the first index of an array is always 0), as shown in the following line of code: var x = 0; 2.The second statement is the condition. As long as the number in the variable x is less than the length of the array, the loop will keep running. This will allow the loop to execute one time for each item in the array. We use the length built-in array property on our shapes array to express that condition, as shown: x < shapes.length; 3.The final statement is what allows the array to iterate over each item. Since indices is what allow the array to iterate over each item. Since indices follow a predictable pattern, we know the number increases by 1 each time. The code x++ is a shortcut for x+1 Now everytime the code loops, x will increase by 1.
  • Commenting out code makes it inactive without deleting it: /* before block of code and */ after.
  • At the most basic level, a function is a set of instructions that tells the computer how to perform a certain task.
  • A function is different from a for loop or conditional statement because functions do not automatically execute when the JavaScript file is run. Instead, you must call a function by name in order to execute it.
  • To create a function, we use the function keyword, similar to how we used the var keyword to declare a new variable. Then we will give the function a unique name and use that whenever we call the function.
  • A functions name is followed by parentheses ()
  • Math functions selects a random number.

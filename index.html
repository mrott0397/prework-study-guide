HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • HTML is used to create the structure of a webpage. It is made up pf elemts which work together to provide content
  • HTML elements are usually composed of an opening and closing tag, represented with angle brackets
  • Elements included within the head are not visible when a user looks at a webpage
  • All the elements that we want the user to see and interact with must be placed inside the body element
  • DOCTYPE html is the Decloration at the top. This decloration tells the browser what document type it should expect when opened
  • html elements are case sensitive
  • "html lang="en" is the start of the document and telling the broswer the language is English (en) This is an attribute
  • An HTML attribute defines additional information about an element. HTML attributes are always defined in the beginning element, not the closing element
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata such as character set, description, keywords, author and viewport
  • "meta charset= "UTF-8" The meta elemt defines the charset, or character set, as UTF-8, which allows us to use most characters and symbols
  • The title element sets the title of the webpage
  • An HTML develepor is not only responsible for providing the structure for thr webpage, but also making sure that it provides a positive user experience
  • Elements that are nested inside other elements are called CHILD ELEMENTS. If there are other elements nested inside of a body element, those three nested elements are known as child elements
  • The element that holds all the children elements, is the PARENT ELEMENT
  • In Semantic HTML Elements, the name of the element describes the content and the roll it plays in a webpage
  • The p element represents a paragraph or block of text
  • The ul element represents Unordered Lists
  • The ol element represents Ordered Lists
  • The li element represents List Items
  • The img element can contain various attributes; the src attribute is the most important because it defines the location of the image
  • The br element creates a line of empty space, or line break
  • The a element (which stabds for anchor) creates links to the same webpage or others. The href attribute points to the url for the link
  • The alt attribute contains a text string that describes the apperance and functionality of the image. The alt text is important for screen readers and other assistive technologies. Alt text can also improve search engine rankings
  • The footer element appears at the bottom of the page and usually contains the author, copywrite, contacts, site map, and navigation
  • Section elements are semantic elements that define a section in a document that contains thematic content
